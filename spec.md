# Specification

## Summary
**Goal:** Build "Sole Street," a streetwear-inspired shoes e-commerce website with product browsing, filtering, detail pages, and a shopping cart.

**Planned changes:**
- Homepage with a hero section (tagline + CTA), featured products grid, and navigation bar with logo, collection links, and cart icon
- Product listing page with responsive grid, category filters (Sneakers, Boots, Sandals, Formal), and sort options (price low-high, high-low, newest)
- Product detail page showing images, name, description, available sizes, price, size selector, and Add to Cart button
- Shopping cart sidebar/page with item list, quantity controls, remove option, line totals, subtotal, and session persistence
- Backend seeded with at least 12 shoe products across at least 3 categories, each with id, name, description, price, category, sizes, and imageFilename; query endpoints for all products, by category, and by id
- Bold modern streetwear visual theme: dark charcoal backgrounds (#1a1a1a), off-white text (#f5f0eb), amber/orange accents (#e8891a), clean sans-serif typography applied consistently across all pages

**User-visible outcome:** Users can browse a stylish shoe catalog, filter and sort products, view product details, select sizes, add items to a cart, and review their cart with quantity controls and a subtotal — all within a cohesive dark streetwear aesthetic.
